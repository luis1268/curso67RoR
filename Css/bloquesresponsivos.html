@media queries

CSS usa la regla de @media al detectar información sobre el dispositivo que esta siendo usado al acceder al sitio web. Por ejemplo, puede ser usado para detectar el tamaño del dispositivo, las capacidades y orientación. Para este curso, nosotros estaremos usandolo para detectar el tamaño del dispositivo, pero teniendo en mente esta capacidad de mucho más.
