JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety of reasons. It is a small advertisement lightweight language allowing maximum flexibility for delevopers to take it in a bunch od different directions. JavaScript lives inside a host enviroment (a web browser or Node server), it can be connected to the objects of these enviroments to providfe programmatic control Over them.

  • Penal: es un tiro que se ejecuta desde los 12 pasos desde la linea del portico, el jugador debe esperar a que el arbitro pite la orden de ejecucion.

  • Cornel: es un tiro que se ejecuta desde una de las 4 esquinas de la cancha, el jugador debe esperar a que el arbitro pite la orden de ejecucion.

  • Tiro libre: es un tiro que se ejecuta desde cualquier lugar de la cancha menos en las areas de meta de la cancha, el jugador debe esperar a que el arbitro pite la orden de ejecucion.

  • Lateral: es saque que se realiza con las manos desde los lados de la cancha, el jugador no necesita esperar a que el arbitro pite la orden de ejecucion.
